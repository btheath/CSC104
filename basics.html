CSC104 HTML Basics

HTML stands for HyperText Markup Language. HTML provides the structure of the page by marking up the text. You can use a variety of HTML tags to structure and organize a webpage. HTML does not really have any rules, but there are best practices that should be used when building the webpage.

History of HTML

  • HTML 1.0 - developed in the early 1990's
  • HTML 2.0 - released in 1995
  • HTML 3.2 - released in 1997
  • HTML 4.01 - released in 1999
  • HTML 5 - current version - released in 2014

Cascading Style Sheets (CSS)

CSS is a language that is used to make HTML look better. CSS can control the layout, colors, fonts - basically any visual element on the webpage can be changed from the default with CSS.

History of CSS

  • CSS1 released in 1996
  • CSS2 released in 1998
  • CSS3 - current version - released in 2011 - living standard

Basic Structure of CSS

                selector{
                    property: value;
                }

CSS Selectors

  • HTML element
  • ID of an HTML element - #id
  • Custom class - .classname

3 Ways to Create CSS

  • External - a separate CSS file- example, main.css
  • Embedded - style tags in the head section
  • Inline - set up CSS on individual HTML elements
